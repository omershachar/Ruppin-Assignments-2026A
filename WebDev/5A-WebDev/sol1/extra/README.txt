תיקיית בדיקות - Extra

קבצים בתיקייה זו:

1. sample-orders.json
   - קובץ JSON המכיל 25 הזמנות לדוגמה
   - ההזמנות מתפרסות על פני יותר משני חודשים (אוקטובר-דצמבר 2024)
   - ניתן לטעון את הקובץ דרך מסך ההזמנות

2. menu-items.json
   - קובץ JSON המכיל את כל הפריטים בבית הקפה
   - מאורגן לפי קטגוריות: משקאות, מאפים, סלטים, כריכים, טוסטים, ארוחות בוקר
   - כל פריט כולל שם ומחיר

הוראות שימוש:
- לטעינת הזמנות מהקובץ sample-orders.json, יש להשתמש בפונקציונליות טעינת קובץ במסך ההזמנות
- הקובץ menu-items.json נטען אוטומטית בעת טעינת האפליקציה
